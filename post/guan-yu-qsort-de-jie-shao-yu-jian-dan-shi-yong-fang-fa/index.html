# 关于qsort的介绍，与简单使用方法

2023-09-12

关于qsort的介绍，与简单使用方法

qsort函数介绍：

1.qsort是 C 标准库 -<stdlib.h>中的一个库函数

				### 2.qsort的声明：

​

void qsort(void *base, size_t nitems, size_t size, int (*compar)(const void *, const void*))

3.参数解析：

  • base -- 指向要排序的数组的第一个元素的指针。
  • nitems -- 由 base 指向的数组中元素的个数。
  • size -- 数组中每个元素的大小，以字节为单位。
  • compar -- 此为比较函数的函数指针，用以决定排序的顺序。注意：如果两个元素的值是相同的，那么它们的前后顺序是不确定的。
  • **void **:我们无法用一个实际类型来代表所以的类型，就将所类型都强制转换为void，用以接受所有实际类型

4.*（重要）compar参数：

int compar(const void *p1, const void *p2);

compar是一个指向一个比较两个元数大小的函数，它是qsort函数排序的核心内容，qsort函数根据compar函数的返回内容来进行排序。

注意两个形参必须是const void *型，在compar函数内部，会将const void *型转换为实际类型

compar函数：

int compare (const void * a, const void * b)
{
	return  ( *(MyType*)a -  *(MyType*)b );
}

注：MyType是你需要排序的对象的类型

如果 返回大于0，则 a所指向的内容大于b的

如果 返回等于0，则 a所指向的内容等于b的

如果 返回小于0，则 a所指向的内容小于b的

代码实现：

int 类型：

int cmpfun_char(const void* a, const void* b)
{
    //升序
	return *(char*)a- *(char *)b;
    //降序
    //return *(char*)b- *(char *)a;
}

qsort(arr, 5, sizeof(int), cmpfun_int);

字符串类型:

int cmpfun_char(const void* a, const void* b)
{
	return *(char*)a- *(char *)b;
}

qsort(arr1, 5, sizeof(char), cmpfun_char);

结构体类型：

struct student {
	int id;
	char name[100];
};
//结构体类型的compar函数可以根据结构体成员来选择排序，这里我就选择了其中的int类型来进行排序，同理也可以选择字符串来进行。
int cmpfun_struct(const void* a, const void* b)
{
	return  (*(struct student*)a).id - (*(struct student*)b).id;
}

struct student xxn[3] = { {12346,"xiaow"},{54646,"xiaohu"},{855455,"xiaog"} };

qsort(xxn, 3, sizeof(arr[0]), cmpfun_struct);
